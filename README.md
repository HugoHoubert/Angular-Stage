In this repository you will find a web application made to learn Angular!

Step 1:

install node and npm and Cloud Foundry 

Step 2:

install angular/cli :
               npm i @angular/cli

command to use with angular/cli

ng generate component "name
ng serve --open
ng build

Step 3: 

put your project in production .
